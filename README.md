## Usage

|                             |               |
|-----------------------------|:-------------:|
| **Launch**                  |               |
| To launch Stacks            | `⌘+Space`     |
|                             |               |
| **Navigation**              |               |
| Move down / up              | `⌘+n` / `⌘+p` or arrow keys |
| Move left / right           | `⌘+h` / `⌘+l` or arrow keys |
|                             |               |
| **Item Manipulation**       |               |
| Move an item down/up        | `⌘+down` / `⌘+up` |
| Bring current stack to the top | `⌘+t`      |
